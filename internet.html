The Internet.

How does the Internet work? What are some daily practices? and how can we implement them into our daily lives?

✱What is Encoding?

Encoding is the process which creates a secure password for your data, whether that be a photo, text, or payment information. You can be sure to thank the encoding process that your phone number isn't leaked, and your credit cards aren't maxed out! Different formats like ASCII or UTF-8 exist for readability across different systems. Data encoding, such as Base64 or URL encoding, transforms binary or special characters into text suitable for transmission.

✱What is Encryption?

Encryption is the process of converting data into a coded format to prevent unauthorized access. It uses algorithms and keys to transform readable data (plaintext) into an unreadable form (ciphertext). Only individuals or systems with the correct decryption key can convert the encrypted data back to its original format.

✱How does Data Travel?

Data travels across networks by first being broken into small packets, each containing a portion of the original information and metadata like sender and receiver addresses. These packets are transmitted through various network paths using routers and switches, following protocols like TCP/IP to ensure proper routing and error handling. If the data is encrypted, it remains unreadable during transmission and is only decrypted at the destination using the correct key.

✱Secure Communication practices.

Secure communication practices are crucial for protecting sensitive data from unauthorized access and attacks. Key practices include encrypting data both in transit and at rest to prevent eavesdropping, using multi-factor authentication to verify user identities, and implementing strong, unique passwords stored securely. Regular software updates help protect against vulnerabilities, while using secure networks, such as encrypted Wi-Fi and VPNs, safeguards data on public connections.

✱HTTPS and Security.

HTTPS (Hypertext Transfer Protocol Secure) is a protocol used to secure communication over the internet by encrypting the data exchanged between a user's browser and a website. It uses SSL/TLS (Secure Sockets Layer/Transport Layer Security) protocols to create an encrypted connection, preventing third parties from intercepting or tampering with the data during transmission. HTTPS ensures that the data sent, such as passwords, payment information, or personal details, is encrypted and secure from potential hackers.

✱Best Practices.

When surfing the web, it's essential to prioritize security and privacy by following best practices. Always look for HTTPS in the URL to ensure encrypted communication, and enable two-factor authentication for accounts that support it. Use strong, unique passwords for each site, and consider a password manager to securely store them. Avoid using public Wi-Fi for sensitive transactions unless you're using a VPN, and stay vigilant against phishing scams by verifying the legitimacy of emails and links. Regularly update your browser and software to patch security vulnerabilities, and consider using ad blockers and privacy tools to minimize tracking.
